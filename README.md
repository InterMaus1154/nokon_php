# Nokon PHP Alpha v1
## Nokon is a lightweight PHP framework - first release

### How to install
`composer create-project nokon/nokon:"1.0.0-alpha" <project_name>`

### How to run
Within project folder:
`php nokon serve`
The app will run by default at `http://127.0.0.1:8000`
